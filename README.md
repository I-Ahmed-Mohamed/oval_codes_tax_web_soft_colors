## 📁 هيكل المشروع

```
oval_codes_tax_web_soft_colors/
├── index.html              # الملف الرئيسي (HTML)
├── css/
│   ├── themes.css         # متغيرات الألوان والـ Dark Mode
│   └── styles.css         # جميع الأنماط والتنسيقات
├── js/
│   ├── utils.js           # الدوال المساعدة والعامة
│   └── main.js            # منطق التطبيق الرئيسي
```

## ✨ المميزات

### 🌙 وضع فاتح وغامق (Light/Dark Mode)
- تبديل سهل بين الوضع الفاتح والغامق
- الألوان تتغير تلقائياً مع الحفظ في localStorage
- يتذكر تفضيلك عند العودة

### 📦 فصل الملفات
- **HTML**: الهيكل الأساسي (index.html)
- **CSS**: منفصل في مجلد `css/`
  - `themes.css`: متغيرات الألوان والـ Dark Mode
  - `styles.css`: جميع الأنماط والتصميم
- **JavaScript**: منفصل في مجلد `js/`
  - `utils.js`: دوال مساعدة وعامة
  - `main.js`: الكود الرئيسي للتطبيق

## 🚀 الاستخدام

1. افتح ملف `index.html` في المتصفح
2. انقر على أيقونة 🌙/☀️ لتبديل الوضع الفاتح/الغامق
3. استخدم الفلاتر والبحث كالمعتاد

## 📝 الملفات الرئيسية

### themes.css
يحتوي على:
- متغيرات CSS لـ Light Mode (`:root`)
- متغيرات CSS لـ Dark Mode (`html.dark-mode`)
- انتقالات سلسة بين الأوضاع

### styles.css
يحتوي على:
- جميع الأنماط والتصميمات
- تنسيقات الـ Dark Mode (باستخدام `html.dark-mode`)
- Responsive design للأجهزة المختلفة

### utils.js
يحتوي على:
- `toggleDarkMode()`: تبديل الوضع الفاتح/الغامق
- `initTheme()`: تهيئة الموضوع عند التحميل
- `updateThemeButton()`: تحديث أيقونة الزر
- دوال مساعدة أخرى (نسخ، البحث، إلخ)

### main.js
يحتوي على:
- منطق التطبيق الرئيسي
- إدارة البيانات والحفظ في localStorage
- عرض المنتجات والأرقام الضريبية
- إدارة النوافذ المنبثقة

## 💾 حفظ البيانات

جميع البيانات تُحفظ تلقائياً في:
- `localStorage.oval_products_v6`: البيانات المنتجات
- `localStorage.oval_clients_v6`: بيانات الشركات
- `localStorage.theme-preference`: تفضيل الموضوع

## 🔄 استرجاع البيانات الأصلية

انقر على زر "استرجاع الأصل" لاستعادة جميع البيانات الأصلية.
